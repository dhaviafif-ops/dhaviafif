FAQ
Pertanyaan Umum
Apakah kamu menerima freelance?

Ya! Saya terbuka untuk proyek freelance web development, AI integration, dan pembuatan aplikasi. Silakan hubungi via email atau WhatsApp.

Berapa lama pengerjaan sebuah website?

Tergantung kompleksitas. Website sederhana 3-7 hari, dengan fitur lengkap bisa 2-4 minggu. Selalu diskusi dulu sebelum memulai.

Teknologi apa yang kamu gunakan?

HTML, CSS, JavaScript, Python, FastAPI, PHP, MySQL, dan berbagai tools modern lainnya sesuai kebutuhan proyek.

Apakah kamu bisa bantu deploy aplikasi?

Ya! Berpengalaman deploy ke GitHub Pages, Railway, dan hosting lainnya. Bisa bantu setup domain dan SSL juga.
